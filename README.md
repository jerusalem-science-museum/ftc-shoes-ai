# shoes-ai

a local node webpage that lets you describe a shoe and using openAI generates an image.

# installation
1. run `chmod +x ./setup.sh && ./setup.sh`
2. add .env file with api key (from [sharepoint folder](https://madaorgil.sharepoint.com/:f:/s/MakeMada/IgD3PsgrSQmFT6F233jOxhgmAaWxE0sYHMhkizUw4fjimoY?e=Lt8mGu)  or by yourself) and replace it with the server/.env
3. reboot.
4. assuming you're running this on 2 screens (with one on top of the other), open display settings and put the relevant screens in the right orientation (i.e. one on top of the other) for the mouse movement.
